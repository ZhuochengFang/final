# Google Scholar 镜像搜索

Google Scholar 直接访问不可用，可通过以下镜像站搜索论文：

## 搜索格式

```
https://scholar.lanfanshu.cn/scholar?q=搜索关键词&hl=zh-CN
```

## 示例

精确匹配标题（用双引号包裹）：
```
https://scholar.lanfanshu.cn/scholar?q=%22An+Energy-Efficient+Comparator+With+Dynamic+Floating+Inverter+Amplifier%22&hl=zh-CN
```

关键词搜索：
```
https://scholar.lanfanshu.cn/scholar?q=floating+inverter+amplifier&hl=zh-CN
```

## 注意

- 镜像站首页可能报错，但直接带搜索参数访问可正常返回结果
- `%22` 为双引号的 URL 编码，用于精确匹配论文标题
- 空格用 `+` 替代
