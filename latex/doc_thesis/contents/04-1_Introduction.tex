\section{绪论}

\subsection{研究背景与意义}

随着物联网终端、生物医学可穿戴设备、边缘智能传感节点等应用快速发展，模数转换器（ADC）与前端模拟接口电路面临“低功耗、高精度、小面积、低电压”并存的设计约束。传统运算放大器（OTA）主导的开关电容电路在追求高增益与高线性时，通常需要较大的静态偏置电流与较高供电电压，这与超低功耗场景的需求存在根本矛盾。尤其在先进工艺下，器件本征增益下降、供电电压持续降低、可用摆幅收窄，使得“以静态功耗换性能”的经典路径越来越难以维持高能效。

动态放大器（Dynamic Amplifier）因“按需放大、近零静态电流”的特征，成为低功耗数据转换系统的重要候选。其基本思想是在离散时间相位内利用电荷再分配、瞬态正反馈或反相器跨导增强等机制完成增益获取，而非依赖持续偏置维持放大状态。该类电路天然适合与开关电容积分器、离散时间噪声整形环路和低带宽高分辨 ADC 架构结合，在微瓦级功耗下仍有机会实现较高信噪失真比（SNDR）与有效位数（ENOB）\cite{wang2021sefiamod,lin2022lvfia,meng2023jssc_mash}。

然而，动态放大器并非“无代价增益”：其性能高度依赖时钟相位、寄生电容、开关注入、器件失配和工艺电压温度（PVT）波动；在低压供电下还会遭遇输入共模受限、输出摆幅不足和非线性加剧等问题。围绕这些瓶颈，学术界提出了多种改进路径，其中以浮动反相放大器（FIA）及其增强形态（如 SEFIA、LVFIA）最具代表性。近年来，相关电平移位（Correlated Level Shifting, CLS）技术进一步提供了在低电压条件下改善线性与能效的新手段，为动态放大器在高性能数据转换中的应用打开了新的设计空间。

\subsection{动态放大器及典型结构发展}

\subsubsection{动态放大器基本机理}

动态放大器通常工作在两个或多个非重叠时钟相位。以典型开关电容前端为例：在采样相位，输入信号被存储于采样电容；在放大相位，电荷通过放大核心与反馈网络重新分配，形成输出增量。与静态 OTA 相比，动态放大器的优势体现在：

\begin{itemize}
    \item 几乎不消耗静态偏置功耗，显著提升能效；
    \item 电路拓扑更易与数字时钟系统协同，便于分相控制；
    \item 在低至中等带宽应用中可获得较高的功耗性能比。
\end{itemize}

其主要挑战包括：

\begin{itemize}
    \item 增益依赖瞬态过程，易受时钟边沿和寄生影响；
    \item 动态节点较多，噪声与注入误差建模复杂；
    \item 线性度与输出摆幅在低电压供电下进一步受限。
\end{itemize}

\subsubsection{FIA：浮动反相放大器}

FIA（Floating Inverter Amplifier）将 CMOS 反相器作为高跨导核心，并通过“浮动”工作方式使其在放大相位内获得较大的等效增益与摆幅利用率。FIA 结构常用于离散时间积分器与噪声整形环路\cite{wang2021sefiamod,lin2022lvfia}，具备以下特征：

\begin{itemize}
    \item 利用反相器高 $g_m/I_D$ 特性，实现低功耗增益获取；
    \item 在开关电容网络中易于实现紧凑面积与较高速度；
    \item 对输入共模与相位时序敏感，需精细设计开关控制。
\end{itemize}

FIA 的关键问题通常不在“能否放大”，而在“放大是否可重复、可预测、可线性化”。因此，许多后续工作围绕其共模控制、动态范围扩展和相位误差抑制展开。

\subsubsection{SEFIA：增强型 FIA}

SEFIA（Super/Switched-Enhanced FIA，文献中命名存在细微差异）可理解为对基础 FIA 的增强版本，其核心目标是提升等效增益与线性，同时控制动态误差累积。常见增强手段包括：

\begin{itemize}
    \item 引入辅助路径提高放大相位有效跨导；
    \item 通过结构对称性与相关采样策略减小失配引入的失真；
    \item 在关键节点进行电平管理，改善大信号条件下的稳定性。
\end{itemize}

SEFIA 的工程价值在于：在功耗增量较小的前提下，提升可用动态范围并降低非线性代价，适合中高精度离散时间 ADC 前端。

\subsubsection{LVFIA：低压浮动反相放大器}

LVFIA（Low-Voltage FIA）面向亚 1.2V 甚至更低供电场景，设计重点从“追求高瞬时增益”转向“在有限电压余量中维持可接受线性”。其核心关注点包括：

\begin{itemize}
    \item 低压下器件过驱动电压不足导致的跨导退化；
    \item 输入/输出摆幅收缩引发的动态范围损失；
    \item 非理想开关效应在低压下占比升高，误差更难忽略。
\end{itemize}

因此，LVFIA 通常需要与共模管理、电平重分配、时钟优化及数字校正协同设计，才能实现真正的系统级能效提升。

\subsection{CLS 技术引入与理论动机}

\subsubsection{CLS 的基本思想}

相关电平移位（CLS）并非简单“抬高或降低直流电平”，而是在时钟相关相位中对关键信号节点进行受控电平平移，使由电平操作引入的误差在后续相位呈现相关性，从而可被部分抵消或显著减弱。其本质是通过“误差相关化”换取“系统可校正化/可抑制化”。

在动态放大器语境下，CLS 通常用于：

\begin{itemize}
    \item 改善低压条件下反相器核心的有效工作区间；
    \item 提升输出摆幅利用率，降低削顶和压缩失真概率；
    \item 减轻因共模漂移导致的增益波动与线性退化。
\end{itemize}

\subsubsection{CLS 与 FIA/LVFIA 的耦合优势}

FIA/LVFIA 依赖离散时间相位完成放大，天然具备“相位可编排”特性，这为 CLS 提供了直接落地平台。二者耦合后可以在不显著增加静态功耗的前提下，改善低压动态放大核心的工作点配置。相比单纯增大器件尺寸或提升时钟驱动强度，CLS 更偏向结构性优化，能够在功耗、线性和面积三者之间形成更均衡的折中。

\subsection{国内外研究现状与趋势}

当前围绕动态放大器及其在 ADC 中应用的研究主线可归纳为三类\cite{wang2021sefiamod,lin2022lvfia,meng2023jssc_mash}：

\begin{itemize}
    \item \textbf{器件与电路级优化}：通过反相器复用、辅助跨导、体偏置等方法提升单位功耗增益；
    \item \textbf{时序与采样级优化}：通过相位重排、误差前馈/反馈、相关双采样降低动态误差；
    \item \textbf{系统级协同优化}：将动态放大器与噪声整形、MASH 架构、数字校正联动设计。
\end{itemize}

近期文献显示，CLS 已从“局部电路技巧”逐步发展为“可系统化的方法模块”，尤其在低压高分辨离散时间 ADC 中表现出较高潜力\cite{hu2022mashcls,meng2023jssc_mash,liu2020pingpongcls,ye2025ctcls}。与此同时，研究焦点也从单一指标最优转向多目标协同：既要提升 SNDR，也要保证鲁棒性、可制造性与版图可实现性。

\subsection{现有问题与研究挑战}

尽管 CLS 与 LVFIA 的结合展现出明显优势，但在工程落地中仍存在若干关键难点：

\begin{itemize}
    \item \textbf{时序敏感性高}：CLS 依赖相位相关性，非重叠时钟误差会直接转化为失真与噪声；
    \item \textbf{寄生与失配影响强}：关键动态节点的寄生电容和不对称布局会削弱理论收益；
    \item \textbf{PVT 鲁棒性不足}：低压下器件特性漂移更显著，需额外容错设计；
    \item \textbf{跨层协同复杂}：电路级优化若未与系统级噪声整形和数字后处理匹配，整体收益有限。
\end{itemize}

因此，研究重点不应停留在“提出新结构”，而应聚焦“结构-时序-系统”联合优化与可验证实现。

\subsection{本文研究内容与技术路线}

针对上述问题，本文围绕“基于 CLS 与 LVFIA 的高能效动态放大器”展开研究，技术路线可概括为：

\begin{itemize}
    \item 建立动态放大器关键非理想因素模型，分析低压条件下主要性能瓶颈；
    \item 研究 CLS 在 LVFIA 中的嵌入方式，明确相位编排与电平迁移策略；
    \item 完成电路级实现与参数优化，重点评估增益、线性、噪声及功耗；
    \item 在系统级场景中验证其对 ADC 整体性能的贡献与适用边界。
\end{itemize}

本文的核心目标是：在维持超低功耗约束的前提下，提升动态放大器在低电压环境中的可用线性与能效表现，为后续高性能低功耗数据转换前端设计提供可复用的方法参考。
