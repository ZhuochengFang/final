# 毕业设计
## 当前完成的情况
- 将CLS运用在了LVFIA上，实现了增益的提升，可以通过仿真明确地看出使用CLS前后，无论是开环/闭环放大器均有明显的增益提升
- 将原先电路中的理想器件（模拟开关、电容）全部进行了替换，全部电路器件均可在TSMC130工艺下设计制造
- 进行了PVT仿真，以及探究了不同反馈电容C2下电路增益的变化
- 完成了毕业论文初稿

## 还没完成的工作和存在的问题

- 版图绘制（电路全部器件均可导入实例，目前正在学习版图绘制）
- 增益提升有限，闭环负反馈SC电路在使用了CLS前后增益的提升最大只有约2倍/6dB，之前研究过闭环正反馈SC电路，能够极大地提升增益，但是正反馈结构可能会带来不知道的问题。
- 部分仿真还未完成

## 个人存在的一些疑问

- 通过阅读多篇论文，发现CLS的运用主要是提高等效的开环增益，降低由于有限增益带来的增益误差（尤其是在电容比C1/C2很小的时候），但是对于本身放大器增益就有限的SC电路，CLS只能将其增益上限提升到2倍/6dB。
- 之前研究过多CLS电容（一个CLS->多个CLS）结构，但是CLS电容超过两个后对增益的提升效果就微乎其微了。
- 有几篇论文的CLS结构甚至互相不一致。
- 开关电容电路的版图中，电容尺寸远远大于MOS管，导致版图布局和布线有困难。是否需要调整用于放大的MOS管尺寸。

# 调试过程

## LVFIA的调试
- 提高各个晶体管的W/L之后，发现单独LVFIA的增益不升反降，比较反常
- nmos1vcap电容模型的bulk接VSS后，电路反而不能正常工作了，只有将bulk接S端短接电路才可以正常工作。
